Equações de 1º grau

Equação é uma igualdade matemática com uma incógnita (valor desconhecido),
ela possui expoente 1 (número em cima) e será igualada a 0 para ser encontrada.
ela será encontrada normalmente na forma ax + b = 0, onde a e b são números reais e x a incognita


Como resolver:
1)Separar os números da incógnita para um lado e os números sem a incógnita para o outro lado
2)Dividir o número que acompanha a incógnita pelo número que

Exemplos:


2x + 4 = 0 → 2x = -4 → x = -2
3x - 9 = 0 → 3x = 9 → x = 3
5x + 10 = 20 → 5x = 10 → x = 2

Equação do 2º grau

Ele será igual a de primeiro grau, porem ela possuira 2 raizes reais.
Ela é formada por um número real (a) multiplicando a incógnita ao quadrado, um número real b multiplicando a incógnita e um número real c sem a incógnita,
onde a é diferente de zero.
Ela é encontrada na forma ax² + bx + c = 0,

Tipos:


Completa: x² + 5x + 6 = 0
Incompleta: x² - 9 = 0
Reduzida: x² - 4x = 0

Como resolver:


1)Fatorar a equação, ou seja, escrever a equação como um produto de dois ou mais fatores
2)Encontrar as raízes, ou seja, os valores de x que tornam a equação verdadeira, igualando cada fator a zero e resolvendo para x

Exemplos:


x² + 5x + 6 = 0 → (x + 2)(x + 3) = 0 → x = -2 ou x = -3
x² - 9 = 0 → (x - 3)(x + 3) = 0 → x = 3 ou x = -3
x² - 4x = 0 → x(x - 4) = 0 → x = 0 ou x = 4

Fórmula de Bhaskara

Ela é usada principalmente para equações do segundo grau que não podem ser fatoradas facilmente.
Sempre possuira 2 raizes reais, podendo ser iguais se o Delta Δ for igual a zero, 2 se for maior que zero, ou nenhuma se for menor que zero.
Ela é dada por:
Δ = b² - 4ac
x = (-b ± √Δ) / 2a
Exemplo:
2x² + 3x - 2 = 0
Δ = 9 + 16 = 25
x = (-3 ± 5)/4 → x = 1/2 ou x = -2

Problemas com equações

Elas são interpratações de frase dadas e assim seram transformadas em equações para serem resolvidas,
elas podem ser de primeiro ou segundo grau, dependendo do problema.

Exemplos:

"Se uma empresa ganha 5 reais por cada produto vendido e tem um custo fixo de 1200 reais, quantos produtos ela precisa vender para ter um lucro de 2000 reais?"
1200 é o custo fixo, 5 é o lucro por produto vendido, e 2000 é o lucro total desejado.
para chegar em 3200 é preciso somar o custo fixo com o lucro total desejado, ou seja, 1200 + 2000 = 3200
3200 = 5x + 1200
