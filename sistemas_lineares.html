Sistemas Lineares

O que são Sistemas Lineares?

Os sistemas lineares são conjuntos formados por duas ou mais equações lineares que possuem incógnitas em comum. O objetivo é encontrar os valores dessas incógnitas que tornam todas as equações verdadeiras ao mesmo tempo.

As equações lineares possuem incógnitas com expoente igual a 1.

Exemplo:
x + y = 2
x − y = 10

Nesse sistema, as incógnitas são x e y.

  • Os números que acompanham as incógnitas são chamados de coeficientes.
  • Os números após o sinal de igualdade são chamados de termos independentes.

Solução de um Sistema Linear

A solução é o conjunto de valores que satisfaz todas as equações simultaneamente.

Sistema:
x + y = 2
x − y = 10

Somando as equações:
2x = 12
x = 6

Substituindo em x + y = 2:
6 + y = 2
y = -4

Portanto, o conjunto solução é:

S = {(6, -4)}

Método da Substituição

Consiste em isolar uma das incógnitas em uma das equações e substituí-la na outra.

Exemplo:
x + y = 7
x - y = 1

Isolando x:
x = y + 1

Substituindo:
(y + 1) + y = 7
2y + 1 = 7
2y = 6
y = 3

x = 3 + 1 = 4

S = {(4,3)}

Método da Adição

Nesse método, somamos ou subtraímos as equações para eliminar uma incógnita.

Exemplo:
x + y = 2
x − y = 10

Somando:
2x = 12
x = 6

Substituindo:
6 + y = 2
y = -4

Método do Escalonamento

O escalonamento transforma o sistema em uma forma mais simples por meio de operações entre as linhas.

Exemplo:

x + y + z = 6
2x + y − z = 3
x − y + z = 2

Utilizando operações entre as equações, eliminam-se incógnitas até que o sistema seja resolvido gradualmente.

Esse método é muito utilizado em sistemas com três ou mais incógnitas.

Classificação dos Sistemas Lineares

Sistema Possível e Determinado (SPD)

Possui apenas uma solução.

D ≠ 0

Exemplo:

x + y = 5
x − y = 1

Sistema Possível e Indeterminado (SPI)

Possui infinitas soluções.

D = 0
Dx = 0
Dy = 0

Exemplo:

x + y = 2
2x + 2y = 4

Sistema Impossível (SI)

Não possui solução.

D = 0
Dx ≠ 0 ou Dy ≠ 0

Exemplo:

x + y = 2
x + y = 5

Interpretação Gráfica

Cada equação do sistema representa uma reta no plano cartesiano.

  • SPD → as retas se cruzam em um único ponto.
  • SPI → as retas coincidem.
  • SI → as retas são paralelas.

Regra de Cramer

A Regra de Cramer resolve sistemas utilizando determinantes. Ela só pode ser aplicada quando o número de equações é igual ao número de incógnitas.

Passo 1: Determinante Principal

Sistema:
x + y = 2
x − y = 10

Matriz dos coeficientes:

|1 1|
|1 -1|

Passo 2: Determinante Dx

Substituindo a primeira coluna pelos termos independentes:

|2 1|
|10 -1|

Dx = (2 × -1) − (1 × 10)
Dx = -12

Passo 3: Determinante Dy

Substituindo a segunda coluna:

|1 2|
|1 10|

Dy = (1 × 10) − (2 × 1)
Dy = 8

Passo 4: Encontrando as incógnitas

x = -12 ÷ -2 = 6
y = 8 ÷ -2 = -4

Logo:
S = {(6,-4)}

Aplicações dos Sistemas Lineares

Os sistemas lineares aparecem em diversas situações do cotidiano.

  • Engenharia;
  • Economia;
  • Administração;
  • Física;
  • Informática;
  • Estatística;
  • Planejamento financeiro;
  • Controle de produção;
  • Distribuição de recursos.

Exemplo do Cotidiano

Uma lanchonete vende hambúrgueres e refrigerantes. Duas pessoas compraram:

2 hambúrgueres e 1 refrigerante = R$ 35
1 hambúrguer e 2 refrigerantes = R$ 25

Utilizando um sistema linear, é possível descobrir o preço de cada produto.

Conclusão

Os sistemas lineares são ferramentas fundamentais para resolver problemas que envolvem várias incógnitas relacionadas entre si. Métodos como substituição, adição, escalonamento e Regra de Cramer permitem encontrar soluções de maneira organizada e eficiente, tornando esse conteúdo essencial para a Matemática e para diversas áreas profissionais.
