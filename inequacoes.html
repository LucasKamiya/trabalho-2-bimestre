Inequações

Inequações são desigualdades matemáticas que possuem uma incógnita.
Em vez de usar "=", usamos sinais como:
> (maior), < (menor), ≥ (maior ou igual), ≤ (menor ou igual)
uma regra muito importante é que quando multiplicamos ou dividimos por um número negativo, o sinal da desigualdade deve ser invertido.

Inequações do 1º grau

São inequações onde a incógnita possui a letra elevada a 1.
Geralmente aparecem na forma: ax + b > 0
onde a e b são números reais e x é a incógnita.

Exemplos:


2x + 4 > 0
3x - 6 ≤ 0
x + 5 > 2

Resolução de inequações

Para resolver, seguimos passos parecidos com equações.


Passos:
1) Isolar a incógnita
2) Se multiplicar ou dividir por número negativo, inverter o sinal

Exemplos:


2x + 4 > 0 => 2x > -4 => x > -2
3x - 6 ≤ 0 => 3x ≤ 6 => x ≤ 2
-2x > 4 => x < -2 (inverte o sinal)

Representação em intervalos

O resultado pode ser escrito como intervalo na reta numérica.

Regras:


] [ => número não incluído
[ ] => número incluído

Exemplos:


x > 2 => ]2, +∞[
x ≥ 3 => [3, +∞[
x < -1 => ]-∞, -1[
