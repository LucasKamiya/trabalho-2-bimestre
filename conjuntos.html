Conjuntos Numéricos

Eles são formas de organizar os números em grupos, de acordo com suas características proprias e aplicações.

Números naturais (ℕ)

Neste conjunto se encontram todos os números inteiros maiores ou iguais a zero, sem incluir os números decimais(com virgula).
Exemplos: 0, 1, 2, 3, 4, 5...

Números inteiros (ℤ)

Pertence a esse conjunto todos os números que não possuem parte decimal, tanto negativos quanto positivos.
Exemplos: -3, -2, -1, 0, 1, 2, 3...

Números racionais (ℚ)

São números que podem ser escritos como fração ou que apresentam casas decimais exatas ou periódicas (que repetem valores infinitas vezes).
Exemplos: 1/2, 3/4, 0,25, 0,333...

Números irracionais

Não podem ser representados por fração e possuem casas decimais infinitas e que não se repetem
Exemplos: √2, π, √3

Números reais (ℝ)

São todos os números que existem na reta numérica.
Incluem os racionais e irracionais.

Relação entre os conjuntos

Sua relação é que cada conjunto pode ser visto como uma extensão do anterior. Onde os naturais são os menos abrangentes e os reais são os mais abrangentes.
ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ

Intervalos numéricos

Os intervalos representam as raízes de uma equação dentro de uma reta numérica.

Tipos de intervalos:


[a, b] => fechado (inclui a e b)| todos os números entre a e b, incluindo a e b pode ser usados para representar soluções de inequações do tipo x ≥ a e x ≤ b
]a, b[ => aberto (não inclui a e b)| todos os números entre a e b, mas não inclui a e b pode ser usado para representar soluções de inequações do tipo x > a e x < b
[a, b[ => inclui só o início| todos os números entre a e b, incluindo a mas não inclui b pode ser usado para representar soluções de inequações do tipo x ≥ a e x < b
]a, b] => inclui só o final| todos os números entre a e b, incluindo b mas não inclui a pode ser usado para representar soluções de inequações do tipo x > a e x ≤ b

Exemplos:


Se Maria tem mais de 18 anos, a idade dela pode ser representada por um intervalo do tipo ]18, ∞[
Se João tem entre 10 e 20 anos, a idade dele pode ser representada por um intervalo do tipo [10, 20]

Representação na reta numérica

Os intervalos podem ser representados com pontos:
● ponto fechado => número incluído [a, b]
○ ponto aberto => número não incluído ]a, b[
