Frações Algébricas

Frações algébricas são frações que possuem letras (variáveis) no numerador e/ou no denominador.
Elas seguem regras parecidas com frações numéricas, mas precisam de mais cuidado
O numero em baixo (denominador) não pode ser zero.

Simplificação

Simplificar significa reduzir a fração dividindo numerador e denominador por um fator comum(dividir por um número ou letra presente em ambos).
Para isso, geralmente fazemos a fatoração.

Exemplos:


(x² - 9) / (x - 3) => (x - 3)*(x + 3) / (x - 3) => x + 3
(2x + 4) / 2 => 2*(x + 2) / 2 => x + 2

Operações com frações algébricas

As operações são feitas de forma semelhante às frações numéricas.

Adição e Subtração:

se não tiver mesmo denominador é necessário iguala-los.


1/x + 2/x = 3/x
se não for igual acha o mmc ,mínimo múltiplo comum, (o menor número que é múltiplo de ambos os denominadores) e depois iguala os denominadores multiplicando o numerador pelo número que multiplicou o denominador para chegar no mcm
1/x + 1/3x => mmc entre x e 3x é 3x, para chegar em 3x multiplicamos o denominador x por 3, então multiplicamos o numerador por 3 também, ficando 3/x + 1/3x = 4/3x

Multiplicação:

Multiplica os valores que estão na parte de cima juntos e os valores que estão na parte de baixo juntos.


(x/2) * (3/x) = x*3 / x*2 = 3/2

Divisão:

Multiplica o de cima com o debaixo em ambas as frações, a nova fração seguira xa ordem da mmprimeira fração.


(x/2) / (3/x) = x*x/2*3 = x²/6

Restrições de domínio

São os valores que a variável NÃO pode assumir, eles normalmente são determinados pela condição de que o denominador não seja zero.
restriçoes tambem podem ser determinadas pelas raizes e pelos enunciados dos problemas, por exemplo, se o problema fala sobre a quantidade de pessoas, a variável não pode assumir valores negativos.

Exemplos:


se temos 3x - 6 e o resultado não pode ser zero
3x - 6 ≠ 0 => 3x ≠ 6 => x ≠ 2
